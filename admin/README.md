# 🛠 CrypInvest (Admin Panel)

This is the internal management dashboard for CrypInvest. It provides full control over users, investment pools, withdrawal requests, and system-wide categories.

## 🚀 Getting Started

1.  **Install dependencies**:

    ```bash
    npm install
    ```

2.  **Environment Setup**:
    Create a `.env` file with the following:

    ```env
    NEXT_PUBLIC_SERVER_URL=http://localhost:8000
    ```

3.  **Run Dev Server**:
    ```bash
    npm run dev
    ```

## 🛠 Features

- **User Management**: Monitor and verify user accounts and KYC statuses.
- **Investment Controls**: Create and manage investment pools and categories.
- **Financial Oversight**: Approve or reject withdrawal and investment requests.
- **Analytics Dashboard**: Get a birds-eye view of platform performance.
