# 🏦 CrypInvest (Investor Frontend)

This is the investor-facing frontend for the CrypInvest platform. Built with Next.js 15, it provides a high-performance, cinematic experience for crypto investors.

## 🚀 Getting Started

1.  **Install dependencies**:

    ```bash
    npm install
    ```

2.  **Environment Setup**:
    Create a `.env` file with the following:

    ```env
    NEXT_PUBLIC_SERVER_URL=http://localhost:8000
    NEXT_PUBLIC_PUBLIC_KEY=your_imagekit_public_key
    NEXT_PUBLIC_URL_ENDPOINT=your_imagekit_endpoint
    PRIVATE_KEY=your_imagekit_private_key
    NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=your_id
    ```

3.  **Run Dev Server**:
    ```bash
    npm run dev
    ```

## 🛠 Features

- Solana Wallet Integration
- Real-time Portfolio Charts
- Investment Pool Explorer
- KYC Verification Flow
- Animated 3D UI (Three.js + Framer Motion)
